# Reverse polish notation calculator

## How does it translate to normal math?

`2 5 +` - `2 + 5`

`4 3 - 5 *` - `(4-3)*5`

## Supported two argument operators
 - \+ addition
 - \- subtraction
 - \* multiplication
 - \\ division
 - ^ exponentiation
 
## Supported one argument operators
 - sqrt square root
 - abs absolute value
 - sin
 - cos
